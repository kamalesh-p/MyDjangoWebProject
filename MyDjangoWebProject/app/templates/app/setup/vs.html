Need to Update Static Files Location During Hosting

wsgi.py

You need not change anything in this

wsgi.py is considered as your application.

application = get_wsgi_application()

This is your application

urls.py

urls.py is considered as your application's URL.

Important datas from django.urls import include

You need to import Include from django.urls to include a new application

urlpatterns = [ path('',include('app.urls')), path('admin/', admin.site.urls),]

You can add the apps which you need to include in your PythonDjangoWebProject/PythonDjangoWebProject/PythonDjangoWebProject/urls.py

Important note : You need to have only one empty path url like this, path('',include('app.urls')) others should have path url like this path('app1/',include('app1.urls')), path('app2/',include('app2.urls')),
