I Promise that You will learn How to Upload Your Project in GitHub in 2 minutes.

Step-1

First SignIn to GitHub

Sign in to GitHub

Step-2

How to create a New Repositories



Step-3

How to create a New Folder.

Click On Create new file.

Create Folder name followed by /

Create a File name.

You can't Create a New Folder without Creating an Empty File. So create a empty New file.

Click On Commit new file to record your changes.



Step-4

How to Upload a file.

Click On Upload files then Click On Choose your files

Select files that You needed.

Click On open

Atlast Click On Commit changes to trecord your changes.



Note:

No Need to Upload Your Python Virtual Environment Folder and other Empty Folders.

Need to Upload Static Folders , __pycache__ Folders and __init__ Files.

In the below diagram Red Marks are Not Uploaded and Green Marks are Uploaded in GitHub.
